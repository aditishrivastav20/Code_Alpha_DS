# 📊 Unemployment Analysis with Python

## 📌 Project Overview
This project analyzes unemployment trends in India using Python. It includes data cleaning, exploratory data analysis (EDA), and visualizations to understand unemployment patterns and the impact of COVID-19.

---

## 🎯 Objectives
- Analyze unemployment data.
- Perform data cleaning.
- Visualize unemployment trends.
- Study the impact of COVID-19.
- Compare Rural and Urban unemployment.

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset
Unemployment in India Dataset

---

## 📈 Visualizations
- Distribution of Unemployment Rate
- Top 10 States by Unemployment
- Correlation Heatmap
- COVID-19 Impact Analysis
- Rural vs Urban Comparison
