# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview
This project is developed as **Task 1** of the **CodeAlpha Data Science Internship**.

The goal of this project is to classify Iris flowers into three different species using the **K-Nearest Neighbors (KNN)** Machine Learning algorithm.

The model predicts whether a flower belongs to:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

---

## 📂 Dataset
- Dataset: Iris Dataset
- Total Records: 150
- Classes: 3
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width

## 🤖 Machine Learning Model

Algorithm Used:
- K-Nearest Neighbors (KNN)

Steps:
1. Split Dataset into Training & Testing Data

3. Predict Test Data
4. Calculate Accuracy
